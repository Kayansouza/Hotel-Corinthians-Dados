# Hotel Nebula

Sistema de gerenciamento hoteleiro utilizando MongoDB.

## Tecnologias
- MongoDB
- JavaScript
- Node.js

## Funcionalidades
- Gestão de hóspedes
- Reservas
- Hospedagens
- Pagamentos
- Avaliações
- Consultas analíticas

## Objetivo
Criar a base de dados para futura API de gestão hoteleira.
